# AIHack MedKid

Workspace cho dự án MedKid.

Ứng dụng chính nằm trong:

```text
medkid-app/
```

Tài liệu phát triển, kế hoạch, quyết định kỹ thuật và script vận hành được đặt trực tiếp trong source code tại `medkid-app/` để AI agent và developer có thể làm việc nhất quán trong repo.

## Lối vào nhanh

```bash
cd medkid-app
npm install
npm run dev
```

Xem thêm:
- `medkid-app/AGENTS.md`
- `medkid-app/docs/requirements.md`
- `medkid-app/docs/architecture.md`
- `medkid-app/docs/plans/`
- `medkid-app/docs/decisions/`
